Java is the choice of language for over 9 million developers across the world. It's suitable for many enterprise applications due to its portability and reliability. I love the object oriented style of programming that Java enforces. I have previously used Java for projects involving Hadoop.

Hadoop is a software framewrk for processing large amounts of data parallelly on a cluster. It uses a Map and Reduce paradigm to split processes into independent chunks which are later combined to give an output. As part of my data mining course in University, I have implemented popular data mining algorithms on a Hadoop platform.

C is the bible of programming languages for any computer science student. I have used C extensively in the study of multithreading, process forking and joining and network programming. I have also used C-like languages (Processing) for developing software on an Arduino.

I like to call myself somewhat of a web developer. I have experience with front-end frameworks such as Bootstrap, Foundation, Semantic UI and Impress.js. I am currently also learning the MEAN Framework - MongoDB, Express.js, AngularJS and Node.js for a full-stack JavaScript experience.
